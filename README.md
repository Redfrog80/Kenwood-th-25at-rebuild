# Kenwood-th-25at-rebuild
Created a battery clam shell so I could replace the dead OEM battery of an old Kenwood handheld radio.
If you wish to replicate the battery using my models, all you need is a cheap usb-c 2S lipo charger, an appropriately sized 2S lipo, and a soldering iron.
If I ever return to this project, I will likely add screw holes so the battery can be dismantled, a physical switch to disconnect the battery from the charge controller (for storage), and a meter to display battery voltage.
